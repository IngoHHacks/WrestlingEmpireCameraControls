# A Mod That Lets You Control The Camera In Wrestling Empire With The Mouse, Keyboard, Or Right Analog Stick

The name says it all. This mod lets you control the camera in Wrestling Empire with the mouse, keyboard, or right analog stick.
Remember all those times the camera was in the wrong place? Well, no more! Now you can move the camera wherever you want it to be.

## Controls
| Action           | Keyboard | Mouse                    | Controller         |
|------------------|----------|--------------------------|--------------------|
| Move Camera      | I J K L  | Right Click + Move Mouse | Right Analog Stick |
| Camera Up / Down | U / O    | N/A                      | N/A                |
| Fast Move        | Shift    | N/A                      | N/A                |
| Zoom In          | =        | Scroll Up                | N/A                |
| Zoom Out         | -        | Scroll Down              | N/A                |
| Lock Distance    | M        | Middle Click             | N/A                |
| Lock Camera      | N        | N/A                      | N/A                |

Note: Since the camera movements override the taunt button, the taunt button has been moved to the Right Shoulder button on controllers.

## Config

The following config options are available:

| Option                  | Type    | Default       | Description                                                                                                                                                                                               |
|-------------------------|---------|---------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| EnableMouseControl      | Boolean | True          | Enables camera control with the mouse                                                                                                                                                                     |
| EnableKeyboardControl   | Boolean | True          | Enables camera control with the keyboard                                                                                                                                                                  |
| EnableScrollControl     | Boolean | True          | Enables camera control with the mouse scroll wheel                                                                                                                                                        |
| EnableControllerControl | Boolean | True          | Enables camera control with the controller right analog stick                                                                                                                                             |
| InvertHorizontal        | Boolean | False         | Inverts the horizontal camera movement                                                                                                                                                                    |
| InvertVertical          | Boolean | False         | Inverts the vertical camera movement                                                                                                                                                                      |
| InvertScroll            | Boolean | False         | Inverts the camera scroll movement                                                                                                                                                                        |
| HorizontalSensitivity   | Float   | 1.0           | The sensitivity of the horizontal camera movement                                                                                                                                                         |
| VerticalSensitivity     | Float   | 1.0           | The sensitivity of the vertical camera movement                                                                                                                                                           |
| ScrollSensitivity       | Float   | 1.0           | The sensitivity of the camera scroll movement                                                                                                                                                             |
| ControllerTauntKey      | String  | RightShoulder | The button to use for taunting with the controller if controller camera controls are enabled. Valid values are: LeftShoulder, RightShoulder, LeftTrigger, RightTrigger, LeftStickButton, RightStickButton |
